# FoodHub Order Analysis Using Python

## Project Overview

FoodHub is a food aggregator company seeking to better understand customer ordering behavior, restaurant demand, and cuisine preferences. This project uses Python-based exploratory data analysis (EDA) to uncover insights that can help improve customer experience and support business growth.

The analysis focuses on identifying trends in restaurant performance, cuisine popularity, customer ordering patterns, and operational opportunities through data-driven decision making.

## Tools & Skills Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Exploratory Data Analysis (EDA)
- Univariate Analysis
- Bivariate Analysis
- Data Visualization
- Business Recommendations

## Business Objectives

- Understand customer ordering behavior
- Identify the most popular cuisines
- Evaluate restaurant demand
- Analyze order characteristics and trends
- Generate actionable business recommendations

## Key Insights

- Identified top-performing restaurants and cuisines
- Evaluated customer demand patterns
- Explored relationships between key variables
- Generated recommendations to improve customer experience and operational efficiency
